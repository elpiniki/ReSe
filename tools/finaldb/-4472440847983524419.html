Browser & System Requirements for BUworks

Browser Requirements

For maintaining stability and security with BUWorks as well as other university systems, Internet Explorer 11 in compatibility mode (Windows 7 or higher required) and Firefox 24 ESR are the recommended browsers. Buworks may work on earlier versions of Firefox and Internet Explorer but our testing is currently limited to these two versions.

  • If you don’t have Firefox 24 ESR installed, it is possible to run the installation package with your current version of Firefox already installed. Instructions are provided to download Firefox 24 ESR whether you are on a Mac or Windows computer.
  • Instructions are also provided for Enabling Internet Explorer 11 Compatibility Mode.

System Requirements

The minimum and recommended system requirements for use of BUworks are comparable to those outlined for Blackboard Learn:

  • Windows Vista SP 2 or Windows 7
  • Mac OS X 10.7 (Lion) or higher

Please note that faculty and staff are eligible to run the most current and complete version of Windows on both BU-owned and home machines, as outlined by the Microsoft Enterprise Agreement. Mac OS X is available through Sourcing.

Troubleshooting

  1. Make sure you have reviewed the Browser and System Requirements outlined above.
  2. Check the version of Java on your computer and update if necessary. Note: Only the Sun/Oracle version of Java is officially supported. If you experience any problems using the IBM version (or others) you should remove that version and install Java, from the link below.
    1. Visit the Java test page.
    2. A box should appear saying, “Your Java is working.” If it does not, please use the Get Help icon at the top of this page to submit a request for assistance.
    3. Check the line titled Version. Your computer should be configured with Java SE 7 Update 51. If you do not have this version of Java installed, please click the Get Help icon at the top of this page.
    4. Make sure that http://www.bu.edu is listed as an exception to the high security level that comes as a default in Java. If you are unsure of how to do this,  please check our knowledge base article or click the Get Help icon at the top of this page for further assistance.
  3. Configure your Pop-up Blocker Settings as described below.
  4. If and ONLY IF you are using a Mac, make sure to update your Adobe Flash Player to the most recent version.
    • You can check your Adobe Flash version and, as needed,
    • Upgrade by downloading from http://get.adobe.com/flashplayer/?promoid=BUIGP

Pop-up Blocker Settings

BUworks requires adjustments to your pop-up blocker settings.
